# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-14

### Added
- Initial release
- Core block letter font with full ASCII character set
- `RenderWord()` and `RenderText()` functions
- Widget with bubbletea integration (implements tea.Model)
- Vim-style Buffer with normal/insert mode editing
- Layout utilities (centering, alignment, word wrapping)
- Styling system with lipgloss integration
- Theme support (DefaultTheme, KartozaTheme)
- Character highlighting for typing games
- Spring-based animation system using harmonica
- WordCarouselAnimator for speed reading apps
- ANSI escape code utilities
- Comprehensive test suite
- Documentation with mkdocs
- Example applications (simple, animated, editor)
